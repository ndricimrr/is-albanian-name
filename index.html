Albanian Name Checker

How to use:

  1. Install the npm package: npm install is-albanian-name

  2. Import the npm package into your project:
    import isAlbanianName from 'is-albanian-name'

  3. Use the function as needed:
    function check(name) { return isAlbanianName(name); }
